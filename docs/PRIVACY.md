# Privacy Policy

Request Mock Lite is a developer tool for capturing API URLs and mocking
responses while debugging web applications.

The extension does not collect, sell, or transmit personal data. Mock rules and
captured request data are stored locally in the browser using Chrome extension
storage. No data is sent to an external server.

Request Mock Lite uses broad host access so it can run on pages selected by the
developer and intercept page-level `fetch` / `XMLHttpRequest` calls for local
debugging.
